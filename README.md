# lq-folder

## Usage

Requires [Node.js](https://nodejs.dev)

```
npm install

npm start
```

## Example output (recursively)

### List

```bash
/Examples/
/Examples/Novartis/
/Examples/Novartis/Novartis A (should be first)
/Examples/Novartis/Novartis B
/Examples/Roche/
/Basic Example
```

### Tree

```bash
root
└─Examples
  ├─Novartis
    ├─Novartis A (should be first).txt
    ├─Novartis B.txt
  └─Roche
└─Basic Example.txt
```
